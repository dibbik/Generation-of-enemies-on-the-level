# Generation of enemies on the level
Генерация врагов на уровне

1. EnemySpawner - Система спавна врагов
Ответственность: Управление процессом создания врагов по времени и выбор точек спавна

Основные функции:
Таймер спавна - отслеживание интервалов между созданиями врагов
Выбор точки спавна - случайный выбор из зарегистрированных точек
Коррекция высоты - автоматическое размещение врагов на поверхности с помощью Raycast
Координация - взаимодействие с пулом объектов для получения врагов

2. EnemyPool - Менеджер пула объектов
Ответственность: Эффективное управление переиспользованием объектов врагов

Основные функции:
Инициализация пула - создание начального количества объектов
Выдача объектов - предоставление неактивных врагов из пула
Возврат объектов - прием врагов обратно в пул для повторного использования
Статистика - отслеживание количества активных/неактивных объектов

Принципы работы:
При запросе врага сначала проверяются неактивные объекты в очереди
При отсутствии свободных объектов создается новый экземпляр
Возвращенные враги деактивируются и сбрасываются в начальное состояние

3. Enemy - Контроллер поведения врага
Ответственность: Управление состоянием и базовой логикой врага

Основные функции:
Управление состоянием - отслеживание активности врага
Координация компонентов - взаимодействие с движением и анимациями
Обнаружение падения - мониторинг позиции для возврата в пул
Инициализация движения - запуск системы перемещения

4. EnemyMovement - Контроллер движения
Ответственность: Физическое перемещение и ориентация врага в пространстве

Основные функции:
Физика движения - применение скорости через Rigidbody.velocity
Поворот в направлении - ориентация модели в сторону движения
Управление гравитацией - сохранение вертикальной составляющей скорости
Остановка движения - сброс velocity при деактивации

Особенности реализации:
Использование FixedUpdate для физически корректного движения
Мгновенный поворот при старте движения для немедленной ориентации
Сохранение гравитации для реалистичного падения с платформ
